### Backend

1. Installations:

- [Install go](https://go.dev/doc/install)

- [Install Docker](https://docs.docker.com/engine/install/)

- [Install entr for live reload](https://github.com/eradman/entr) (optional)

2. Navigate to the server directory:

```bash
cd server
```

```bash
cp .env.example .env
```

```bash
make reset
```
